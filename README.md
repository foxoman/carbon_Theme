# Install

## Windows
copy the .xml file to  `%APPDATA%\QtProject\qtcreator\styles`

## MacOS
copy the .xml file to `~/.config/QtProject/qtcreator/styles/`

## Linux
copy the .xml file to `~/.config/QtProject/qtcreator/styles/`
