# Install

## Windows
`xcopy qdarksky.xml %APPDATA%\QtProject\qtcreator\styles`

## MacOS
`cp qdarksky.xml ~/.config/QtProject/qtcreator/styles/`

## Linux
`cp qdarksky.xml ~/.config/QtProject/qtcreator/styles/`
